安装 Scikit-learn (sklearn) 最简单的方法就是使用 pip 安装它.
在你的终端上执行 (pip install scikit-learn) 就好啦~ 
注意 python3.x版本的用户要使用 (pip3 install scikit-learn). 
还有要确保你有安装过 numpy 和 scipy这两个模块.
