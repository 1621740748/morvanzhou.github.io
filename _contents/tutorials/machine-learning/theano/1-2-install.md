在 MacOS 和 Linux 上安装 theano 都很简单,
只需要用 pip 安装所需的模块,
再安装 pip install theano就好了.
在 Windows 上安装 theano 可能会有点纠结,
你可以参考这个[网站](http://deeplearning.net/software/theano/install_windows.html#install-windows)
进行安装.
